# Proof-Of-Concept ANCS Client for Google Pixel Watch
The fact that this works at all it pretty cool and saves me about $100

## Setup
- You'll need a Pixel Watch that is already activated by an Android device. If you don't have one, you can use an Android Emulator and [Google's Bumble](https://google.github.io/bumble/platforms/android.html) (please star this repo if you'd like me to make a tutorial!)

- In the settings of your Pixel Watch, manually pair with your iPhone.

- Open this project in the latest version of [Android Studio](https://developer.android.com/studio), build and run it on your Pixel Watch.

- On your iPhone you should recieve a prompt asking for permission to share System Notifications with the device, accept this prompt.

- Try texting yourself to test :) 


I've only tested this on my 1st Gen Pixel Watch.
